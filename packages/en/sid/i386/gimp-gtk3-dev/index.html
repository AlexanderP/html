Headers and other files for compiling plugins for GIMP

This package contains the header files for the GNU Image Manipulation Program, along with the static versions of libgimp. It also includes the gimptool-2.0 utility.

Install this package if you wish to compile your own plugins, or if you wish to develop packages that use gimp-devel.
