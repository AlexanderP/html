plugins for Psi+

This package contains plugins for Psi+.

Psi is a cross-platform powerful XMPP/Jabber client (Qt, C++) designed for the Jabber power users. Psi+ is a development branch of Psi IM XMPP/Jabber client by team.

Project Purpose: collection, refinement and writing new patches for transfer them to Psi upstream developers.

Full list of features you can found at:

http://psi-plus.com/wiki/doku.php/en:features
