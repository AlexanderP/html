Web browser for LeechCraft

This package contains a full-featured web browser for LeechCraft based on WebKit. Poshuku is fully extensible with plugins. Currently it features:

  • support for all major web-standards
  • integration with other plugins
  • autodiscovery
  • tagging bookmarks
  • support for SQLite or PostgreSQL storage

Such sub-plugins available in this package:

  • cleanweb (Ad filter for Web browser) This is a plugin for the Poshuku web browser that filters out ads according to predefined rules. It is compatible with Firefox's AdBlock rules list files. It also supports disabling Flash and allowing it to load only after you click on the corresponding button.
  • fatape (GreaseMonkey support layer) The Poshuku FatApe plugin adds support for GreaseMonkey scripts to the Poshuku browser. FatApe also supports basic scripts management, allowing to install or remove them easily.
  • filescheme (file://-schemes support) This is a plugin for the Poshuku web browser that allows it to handle file:// schemes.
  • fua (Fake user agent plugin) This is a plugin for the Poshuku web browser that allows it to set fake user agents for different domains or even URLs based on regexps.
  • keywords (Support of url keywords)
  • onlinebookmarks (Support for syncing bookmarks with online services) This is a plugin for the Poshuku web browser that allows it to synchronize local bookmarks with online social bookmarking systems like Read It Later.
  • pintab (Support of pinned tab)
  • pogooglue (Support for searching selected text in Google) This is a plugin for the Poshuku web browser that replaces default flash-based video players on some sites with any suitable LeechCraft's media player thus avoiding the need for Flash. First you should install package:

leechcraft-provider-mediaplayer

LeechCraft is a free modular "Internet client" application.

LeechCraft allows one to browse the web, read RSS/Atom feeds, download files via BitTorrent, HTTP, FTP and Direct Connect, automatically stream, download or play podcasts and other media files and much more.
