Advanced Notifications framework for LeechCraft

This plugin provides support for the notifications framework of the same name. It supports various notifiers like visual notifications, audio notifications, tray notifications and custom command execution.

It also allows one to define a list of rules for the events to be matched with, thus providing fine-grained notifications configuration for different events. For example, one could make it notify about IM messages from "Personal" group using all available methods, and notify about every message in a chosen conference using only visual notifications.

Advanced Notifications module contains a default set of rules pretty suitable for average use cases.

LeechCraft is a free modular "Internet client" application.

LeechCraft allows one to browse the web, read RSS/Atom feeds, download files via BitTorrent, HTTP, FTP and Direct Connect, automatically stream, download or play podcasts and other media files and much more.
