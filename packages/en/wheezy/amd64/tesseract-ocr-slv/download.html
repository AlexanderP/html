Download Page for tesseract-ocr-slv_3.04.00~git20140812-1_all.deb

If you are running Debian, it is strongly suggested to use a package manager like aptitude or synaptic to download and install packages, instead of doing so manually via this website.

1. Add this line to your /etc/apt/sources.list :

deb http://notesalexp.org/debian/wheezy/ wheezy main

2. Fetch and install the GnuPG key

sudo aptitude update && sudo aptitude install notesalexp-keyring

sudo aptitude update

or

wget -O - https://notesalexp.org/debian/alexp_key.asc | sudo apt-key add -

sudo aptitude update

3. Enjoy

sudo aptitude install tesseract-ocr-slv
