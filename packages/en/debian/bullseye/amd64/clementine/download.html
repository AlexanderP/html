Download Page for clementine

If you are running bullseye, it is strongly suggested to use a package manager like aptitude or synaptic to download and install packages, instead of doing so manually via this website.

1. Open a terminal and install apt-transport-https

sudo apt-get install apt-transport-https

2. Add this line to your /etc/apt/sources.list :

deb https://notesalexp.org/debian/bullseye/ bullseye main

3. Fetch and install the GnuPG key

sudo apt-get update -oAcquire::AllowInsecureRepositories=true
sudo apt-get install notesalexp-keyring -oAcquire::AllowInsecureRepositories=true
sudo apt-get update

or

wget -O - https://notesalexp.org/debian/alexp_key.asc | sudo apt-key add -
sudo apt-get update

4. Enjoy 

sudo aptitude install clementine

You can download the requested file from the clementine subdirectory at this link:

https://notesalexp.org/debian/bullseye/pool/main/c/clementine-qt5/clementine_1.4.0~git906-g5968648aa-1_amd64.deb
