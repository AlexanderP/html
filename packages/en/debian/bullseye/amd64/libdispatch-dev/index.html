development files for the Grand Central Dispatch API

The libdispatch project consists of the user space implementation of the Grand Central Dispatch API as seen in Mac OS X version 10.6 (Snow Leopard).

This package contains the headers, manpages, and other files needed to build programs that use libdispatch.
