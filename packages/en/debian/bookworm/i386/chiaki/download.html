Download Page for chiaki

If you are running bookworm, it is strongly suggested to use a package manager like aptitude or synaptic to download and install packages, instead of doing so manually via this website.

1. Open a terminal and install apt-transport-https

sudo apt-get install apt-transport-https

2. Add this line to your /etc/apt/sources.list :

deb https://notesalexp.org/debian/bookworm/ bookworm main

3. Fetch and install the GnuPG key

sudo apt-get update -oAcquire::AllowInsecureRepositories=true
sudo apt-get install notesalexp-keyring -oAcquire::AllowInsecureRepositories=true
sudo apt-get update

or

wget -O - https://notesalexp.org/debian/alexp_key.asc | sudo apt-key add -
sudo apt-get update

4. Enjoy 

sudo aptitude install chiaki

You can download the requested file from the chiaki subdirectory at this link:

https://notesalexp.org/debian/bookworm/pool/main/c/chiaki/chiaki_2.1.0+git717-8911a44-9reb~2401_i386.deb
