Development files for libmypaint

libmypaint is the brushstroke rendering engine used by MyPaint. It is highly configurable, allowing artists to create brushes that simulate wet paint, dry media, airbrushes, or nonrealistic digital media.

This package contains development headers and pkgconfig files for libmypaint.
