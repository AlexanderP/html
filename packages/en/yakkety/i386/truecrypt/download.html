Download Page for truecrypt_7.1a-4_i386.deb

If you are running Ubuntu, it is strongly suggested to use a package manager like aptitude or synaptic to download and install packages, instead of doing so manually via this website.

1. Add this line to your /etc/apt/sources.list :

deb http://notesalexp.org/debian/yakkety/ yakkety main

2. Fetch and install the GnuPG key

sudo aptitude update && sudo aptitude install notesalexp-keyring

sudo aptitude update

or

wget -O - http://notesalexp.org/debian/alexp_key.asc | sudo apt-key add -

sudo aptitude update

3. Enjoy

sudo aptitude install truecrypt

You can download the requested file from the pool/main/t/truecrypt subdirectory at this link:
