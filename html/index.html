notesalexp.org

notesalexp.org - Unofficial repository for Debian GNU/Linux and Ubuntu. You can find the packages of this repository following this link.

Instructions

1. Open a terminal

sudo nano /etc/apt/sources.list #to open repository file with a text editor

Add this line for Wheezy: deb http://notesalexp.org/debian/wheezy/ wheezy main
Add this line for Jessie: deb http://notesalexp.org/debian/jessie/ jessie main
Add this line for Stretch: deb http://notesalexp.org/debian/stretch/ stretch main
Add this line for Buster: deb http://notesalexp.org/debian/buster/ buster main
Add this line for Sid: deb http://notesalexp.org/debian/sid/ sid main
Add this line for Precise: deb http://notesalexp.org/debian/precise/ precise main
Add this line for Trusty: deb http://notesalexp.org/debian/trusty/ trusty main
Add this line for Xenial: deb http://notesalexp.org/debian/xenial/ xenial main
Add this line for Yakkety: deb http://notesalexp.org/debian/yakkety/ yakkety main
Add this line for Zesty: deb http://notesalexp.org/debian/zesty/ zesty main

Save and close sources.list

2.Fetch and install the GnuPG key

sudo apt-get update && sudo apt-get install notesalexp-keyring #to add gpg key
sudo apt-get update #to update package list

or

wget -O - http://notesalexp.org/debian/alexp_key.asc | sudo apt-key add -
sudo apt-get update #to update package list

3. Enjoy

sudo apt-get install package

Warning: Need to update the gpg-key.(2015-09-20)
Warning: Some packages for Debian GNU/Linux require http://www.deb-multimedia.org/ repository to be connected.
